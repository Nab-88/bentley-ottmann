PROJET D'ALGORITHME ET DE STRUCTURES DE DONNÉES
par Stéphane Dong et Nicolas Bernard Groupe 3 - Ensimag

Ce projet propose une implémentation du célèbre algorithme de Bentley_Ottman
permettant de trouver les intersections entre des segments par "balayage"

Le répertoire /bentley_ottman contient plusieurs fichiers/répertoires:
- bo.py --> module contenant la plupart des fonctions de l'algo de Bentley Ottman
- vivants.py --> module regroupant les fonctions manipulant les vivants
- /geo --> qui est un répertoire contenant des modules de géométrie
- /sortedcontainers --> est un repertoire contenant les modules permettant l'utilisation
des sortedcontainers
- demo_tycat.py --> modules permettant d'afficher des segments et des points dans terminology
